边境走私大亨 · 豪华版

一款基于浏览器的走私模拟经营游戏。你将在东南亚边境地带穿梭，通过低买高卖、投资BTC、招募军队、完成事件链等方式积累财富，成为一代走私大亨。

✨ 核心特色

· 完全本地运行：纯 HTML + JavaScript，不请求任何权限，无需联网，所有数据仅保存在你设备的浏览器中。
· 丰富玩法：多城市市场、黑市交易、比特币合约、军事行动、投资分红、NPC好感度、成就系统……总有新挑战。
· 事件驱动：随机事件与事件链让每次游玩都充满变数。
· 开源可信：代码完全公开，采用 MIT 许可证，欢迎审查、修改、二次开发。

🛡️ 安全声明

本游戏绝不收集任何个人信息，无需任何权限，运行时完全离线，可放心使用。所有存档均保存在你的浏览器本地，你也可以手动导出备份。
